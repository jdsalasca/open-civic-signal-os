# Agent Execution Pack

Execution assets for Gemini/Codex agents implementing Open Civic Signal OS.

## Files

- `user-stories.md`: detailed, acceptance-driven stories.
- `process-flows.md`: lifecycle and operations Mermaid diagrams.
- `system-design.md`: architecture boundaries and data flow contracts.
- `execution-checklists.md`: implementation, PR, and release checklists.

## Operating Rule

No implementation starts without a Story ID and explicit acceptance criteria.

- docs/agent/operations-runbook.md: execution standard runbook.
- docs/agent/sprint-execution-model.md: weekly multi-agent operating model.
